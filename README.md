<h3>Ruby 3.3.6 / Rails 8.1.1, Vite v7.2.2 + Vue v3.5.24 and PostgreSql Start-up Project</h3>

<p>Dependencies</p>
<p>1. Node 23.5.0</p>
<p>2. Npm 11.1.</p>

<p>Requirements: </p>
<p>1. Postgres Management</p>
<p>2. Vite + Vue - Typescript</p>
<p>3. Install Google or Microsoft Authenticator in your Mobile Phone</p>

<p>FEATURES:</p>
<p>1. User Login, Logout and Account Registration</p>
<p>   JWT Authorization and Authentication</p>
<p>2. Multi-Factor Authenticator (MFA)</p>
<p>3. User Profile, Change Password, Enable/Disable MFA <br/>
This will require your Google/Microsoft Authenticator</p>
<p>4. Upload User's Profile Picture </p>
<p>5. Products Listings, Catalogs pagination and Product Search</p>
